# cube

### setting up the development enviroment.

- npm install
- **grunt** ( build app start & start watching for changes )

### starting app from commandline.

- npm start
- electron .
- nodemon -x electron .

### publishing

  https://www.npmjs.com/package/electron-builder
  example-cmd: node_modules/.bin/build --win
